# الأمن السيبراني في عصر الذكاء الاصطناعي

عرض تقديمي تفاعلي مبني بـ React يغطي تأثير الذكاء الاصطناعي على الأمن السيبراني — من الهجمات المتقدمة إلى آليات الدفاع الحديثة.

## المميزات

- **14 سلايد تفاعلي** — مع أنيميشن وتنقل سلس بالكيبورد والماوس
- **تجربتان عمليتان** — LLM Excessive Agency و Prompt Injection
- **تصميم مستوحى من Claude Code** — إطار بكسل مميز مع خلفية بيج فاتحة
- **متجاوب بالكامل** — نسبة 16:9 تعمل على أي شاشة وبروجكتر
- **عربي بالكامل** — مع خطوط عربية احترافية محسّنة للقراءة

## التقنيات

| التقنية | الاستخدام |
|---------|-----------|
| React 19 | واجهة المستخدم |
| Framer Motion | الأنيميشن |
| Vite 5 | البناء والتطوير |
| React Router | التنقل بين الصفحات |

## التثبيت والتشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل بيئة التطوير
npm run dev

# بناء ملفات الإنتاج
npm run build

# معاينة ملفات الإنتاج
npm run preview
```

## النشر على Vercel

1. ارفع المشروع على GitHub
2. افتح [vercel.com](https://vercel.com) وسجل دخول بحساب GitHub
3. اضغط **Import Project** واختر المستودع
4. الإعدادات تلقائية (Vite يُكتشف تلقائياً)
5. اضغط **Deploy**

> ملف `vercel.json` يتعامل مع client-side routing تلقائياً.

## النشر على GitHub Pages

```bash
npm run build
```

ثم ارفع محتويات مجلد `dist/` إلى فرع `gh-pages`.

```
src/
```
